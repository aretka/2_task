# 2uzduotis

## Versija v0.5

 Versijoje v0.5 išmatuojamas nuskaitomų duomenų laikas bei tų duomenų skirstymas į dvi dalis naudojant list, vector bei deque duomenų tipus. Rezultatai rodo, kad vector bei deque panaudoja praktiškai tiek pat laiko perskaityti failus bei juos suskirstyti į dvi grupes, o naudojant list duomenų skaitymas užtrunka kiek mažiau, tačiau šių duomenų suskirstymas į dvi gruper trunka ilgiau.
